Getting Started

If you have not done so, please install the ARCS software by following installation instructions.

Environment Set-Up

Before running ARCS applications, you must first set up environment vairables.


For Windows platform, please run in a command window the following command 
1
 
$  configureARCS.bat
For Mac OS X platform, please run in a terminal the following command 
1
 
$  source /usr/local/arcs-1.0/bin/envs.sh

Try it out

A good place to start is first to watch the installation and usage demo video, and then follow the LRMECS reduction tutorial to do a data reduction of a sample LRMECS data file.
